Quick JS Injection

For testing fun JS scripts!

Step 1: Make sure the Chrome Bookmarklet bar is shown

Step 2: Adding the bookmarks

For each below, right click on the bookmarks bar, and click 'Add Page'. Then enter the Name and URL. 

Name: Inject New URL
URL: javascript:(function()%7B(function()%20%7Bwindow.__JS_bookmarklet_clear_last_url%20%3D%20true%3B(function(d%2C%20t)%20%7Bwhile%20(t.length)%20%7Bvar%20s%20%3D%20d.createElement(%22script%22)%3Bs.src%20%3D%20t.shift()%3Bd.head.appendChild(s)%3B%7D%7D)(document%2C%5B%22https%3A%2F%2Fs3.amazonaws.com%2Fjs-bookmarklet-crossdomain-storage%2Flast_url_bookmarklet.js%22%5D)%7D)()%7D)()   


Name: Inject Last URL
URL: javascript:(function()%7B(function()%7B(function()%7B(function(d%2Ct)%7Bwhile(t.length)%20%7Bvar%20s%3Dd.createElement(%22script%22)%3Bs.src%3Dt.shift()%3Bd.head.appendChild(s)%3B%7D%7D)(document%2C%5B%22https%3A%2F%2Fs3.amazonaws.com%2Fjs-bookmarklet-crossdomain-storage%2Flast_url_bookmarklet.js%22%5D)%7D)()%7D)()%7D)()

The above bookmarklet will inject the last url you injected. So if you refresh the page, or go to a different page, you don't have to re-enter the javascript file url.
