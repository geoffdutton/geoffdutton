Make sure the Chrome Bookmarklet bar is shown

Right click and add each link as a bookmarklet

This will prompt you for a new javascript file to inject.

This will inject the last url you injected. So if you refresh the page, or go to a different page, you don't have to re-enter the javascript file url.
